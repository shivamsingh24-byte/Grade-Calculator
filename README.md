# Student Grade Calculator in Python

## Project Description
This project is a simple Python program that takes student marks as input and assigns grades based on the grading system. It also provides encouraging messages based on performance.

---

## Features
- Takes student name and marks as input
- Uses if-elif-else statements for grading
- Input validation using while loop
- Error handling using try-except
- Uses function for reusable grading logic
- Displays encouraging messages

---

## Grading Logic

| Marks Range | Grade |
|-------------|--------|
| 90 - 100    | A      |
| 80 - 89     | B      |
| 70 - 79     | C      |
| 60 - 69     | D      |
| 0 - 59      | F      |

---

## How to Run the Program

Make sure Python is installed in your system.

Run the following command:

python grade_calculator.py
